Class

SqrOsc

Name

set()
Examples
import processing.sound.*;
SqrOsc square;

void setup() {
  size(640, 360);
  background(255);
    
  // Create square wave oscillator.
  square = new SqrOsc(this);
  square.play();
}      

void draw() {
}

void mousePressed() {
  float freq=200;
  float amp=0.5;
  float add=0.0;
  float pos=1;
  square.set(freq, amp, add, pos);
}
Description Sets amplitude, add and pan position with one method.
Syntax
.set(freq, amp, add, pos)
Parameters
freq float: The frequency value of the oscillator in Hz.
amp float: The amplitude of the oscillator as a value between 0.0 and 1.0.
add float: Offset the output of the oscillator by given value
pos float: The panoramic position of the oscillator as a float from -1.0 to 1.0.
Returnsvoid
